Visão geralOverview

O SpotCredit calcula a prestação mensal de um crédito habitação a partir de taxas reais e públicas dos principais bancos portugueses, e mostra a capacidade de endividamento, os limites regulatórios e a poupança das medidas de apoio jovem em vigor.

SpotCredit calculates the monthly payment of a Portuguese mortgage from real, publicly-published rates of the main banks, and shows borrowing capacity, regulatory limits and the savings from the youth support measures currently in force.

É deliberadamente uma página estática de ficheiro único: todo o HTML, CSS e JavaScript vivem dentro de index.html. Não há servidor aplicacional, base de dados, nem framework. O utilizador introduz rendimento, valor do imóvel, entrada e prazo; tudo o resto é calculado no browser e nada sai do dispositivo.

It is deliberately a single-file static page: all HTML, CSS and JavaScript live inside index.html. There is no application server, no database, no framework. The user enters income, property value, deposit and term; everything else is computed in the browser and nothing leaves the device.

Porquê tão simples?Why so simple? A simplicidade é uma feature, não uma limitação. Ficheiro único significa: deploy estático em qualquer lado, edição instantânea, e zero risco de supply-chain (sem node_modules a auditar). Propostas de "modernizar" para React/Vite/bundler são rejeitadas por princípio. Simplicity is a feature, not a limitation. Single-file means: static deploy anywhere, instant edits, and zero supply-chain risk (no node_modules to audit). Proposals to "modernise" to React/Vite/bundler are rejected on principle.

Bancos cobertosCovered banks

CGD, ABANCA, Millennium BCP e Bankinter — em taxa variável, mista e fixa. As taxas são introduzidas manualmente por commit (ver §12); nunca são obtidas por scraping.

CGD, ABANCA, Millennium BCP and Bankinter — across variable, mixed and fixed rates. Rates are entered manually by commit (see §12); never scraped.

Modelo de dados — BANKSData model — BANKS

BANKS é um objecto com três arrays — variavel, mista e fixa. Cada entrada é um produto de crédito de um banco. A convenção de nome é Banco — variante (o sufixo após o travessão identifica o produto, não o banco).

BANKS is an object with three arrays — variavel (variable), mista (mixed) and fixa (fixed). Each entry is a credit product from a bank. Naming convention is Bank — variant (the suffix after the dash identifies the product, not the bank).

    // … 5 mais / 5 more
  ],
  mista: [
    { nome: 'Bankinter — mista 2 anos', idx: 12, spread: 0.70, tan_fixa: 2.25, taeg: 4.0, bonus: 'TAN 2,25% fixo 24m → Euribor 12m + 0,70%' },
  ],
  fixa: [
    { nome: 'Millennium BCP — fixa c/ bonif.', idx: 0, spread: 0, tan_fixa: 4.05, taeg: 4.7, bonus: 'domiciliação + seguros Ocidental' },
  ]
};

A prestação — método francêsThe payment — French method

A prestação mensal usa o sistema de amortização francês (prestação constante), o standard português:

The monthly payment uses the French amortization method (constant payment), the Portuguese standard:

P = C × i / (1 − (1 + i)^−n)

C = capital em dívida / outstanding principal
i = taxa mensal = (Euribor + spread) / 12 / 100
n = número de meses = prazo × 12

// implementação / implementation
function calcP(capital, tan, meses) {
  const r = tan / 100 / 12;
  if (r < 0.00001) return capital / meses;  // taxa ~0
  return capital * r * Math.pow(1+r, meses) / (Math.pow(1+r, meses) − 1);
}
Taxa de esforço (DSTI)Effort rate (DSTI) É a fatia do rendimento líquido absorvida pelas prestações: (prestação + outros créditos) / rendimento. O simulador pinta verde até 35%, amarelo até 45% e vermelho acima — alinhado com a recomendação do BdP de 35–40%. The share of net income absorbed by repayments: (payment + other loans) / income. The simulator shows green up to 35%, amber up to 45% and red above — aligned with the BdP recommendation of 35–40%.

IMT — Imposto Municipal sobre Transmissões (HPP, continente)— property transfer tax (primary residence, mainland)

A tabela IMT_HPP tem escalões com taxa e parcela a abater. calcIMT(v) percorre os escalões: IMT = valor × taxa − parcela. Actualizar todos os Janeiros conforme a Portaria da AT. A tabela dos Açores/Madeira é diferente — actualmente só cobrimos o continente.

The IMT_HPP table has brackets with a rate and a deductible portion. calcIMT(v) walks the brackets: IMT = value × rate − portion. Update every January per the AT's official ordinance. The Azores/Madeira table differs — we currently cover only the mainland.

Lógica jovem & não-residenteYouth & non-resident logic

Dois toggles alteram o cálculo e são mutuamente exclusivos: ligar "não-residente" desliga e bloqueia o painel jovem, porque as medidas de apoio exigem residência fiscal em Portugal.

Two toggles change the calculation and are mutually exclusive: turning on "non-resident" disables and locks the youth panel, because the support measures require fiscal residency in Portugal.

Garantia Pública — financiamento a 100%Public Guarantee — 100% financing

Sub-toggle dentro do painel jovem. Quando activo, o Estado é fiador até 15% do valor e o capital passa a ser o valor total do imóvel (LTV 100%, sem entrada). Avisos automáticos se: imóvel > 450.000 €, rendimento > 5.800 €/mês, ou não-HPP.

Sub-toggle inside the youth panel. When on, the State guarantees up to 15% of the value and capital becomes the full property value (100% LTV, no deposit). Automatic warnings if: property > €450,000, income > €5,800/month, or non-primary-residence.

Regras absolutasHard rules

Taxas: manuais, nunca scrapedRates: manual, never scraped

As taxas são actualizadas por commit. Não implementar scraping de sites de bancos — é frágil, viola ToS, e cria falsa precisão (os spreads negociados são sempre diferentes do publicado). Única excepção: a Euribor, de fonte oficial estável.

Rates are updated by commit. Do not scrape bank websites — it is fragile, violates ToS, and creates false precision (negotiated spreads always differ from published ones). Sole exception: Euribor, from a stable official source.

Disclaimers legais obrigatóriosMandatory legal disclaimers

A página mantém sempre visível: "Os valores são meramente indicativos e baseados em informação pública. Não constituem aconselhamento financeiro nem proposta de crédito." Não remover, não esconder, não diminuir.

The page always keeps visible: "Values are indicative only and based on public information. They are not financial advice nor a credit proposal." Do not remove, hide or shrink it.

Não é um intermediário de créditoNot a credit intermediary

O SpotCredit não está registado no Banco de Portugal. Logo: não pode dizer "o melhor crédito para si é X", não pode encaminhar leads com comissão, não pode aceitar dados pessoais para análise.

SpotCredit is not registered with Banco de Portugal. Therefore: it cannot say "the best loan for you is X", cannot route commissioned leads, cannot accept personal data for analysis.

PrivacidadePrivacy

100% client-side. Nenhum dado do utilizador sai do browser. Sem analytics que capturem inputs. Plausible/Umami para pageviews agregados é aceitável; tudo o resto, não.

100% client-side. No user data leaves the browser. No analytics that capture form inputs. Plausible/Umami for aggregate pageviews is acceptable; nothing else.
